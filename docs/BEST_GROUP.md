future-ventures track. This document is the canonical reference for the group
hierarchy; the `nirva.sell` codebase in this repository is one leaf within it
(see [Where nirva.sell fits](#where-nirvasell-fits)).

## Where nirva.sell fits

This repository (`nirva.sell`) is the online-seller operating system вҖ” stock,
orders, profit, promotions, live selling, invoicing. In the group hierarchy it
sits under:

```
рҹ’» NIRVA
в””в”Җв”Җ NirvaCore (ERP)
    в”ңв”Җв”Җ CRM          вҶҗ customers, RFM, loyalty, segments
    в”ңв”Җв”Җ Accounting   вҶҗ P&L, cash flow, expenses, tax invoices
    в””в”Җв”Җ Document     вҶҗ invoices, tax invoices, exports
```

It also touches **NirvaTrade** (commerce / marketplace selling) and consumes
**NOVA** capabilities (Nova Vision for product imaging, Nova Voice, AI agents
for generation and analytics) through the app's BYOK AI features.
