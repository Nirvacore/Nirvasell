## The funding ladder
```
Month 0   Consulting (Diagnosis)         → cash now, no product needed
   │       + sell Blueprints              → high-margin productized IP
   ▼
Month 1-3 CHECK at BEST + 2-3 referrals  → first recurring revenue
   │       "Bring Your Mess" migration removes the adoption barrier
   ▼
Month 3-6 EXECUTIVE upsell               → premium ARPA, board-level buyer
   ▼
Month 6-12 PROCURE + COMPLIANCE          → biggest ROI, stickiest, defensible
   ▼
Year 2+   Marketplace + full SaaS        → compounding, network effects
```

## Pricing logic
- **CHECK** — per-active-worker/month (scales with the customer's workforce; BEST alone = 1,356 seats).
- **EXECUTIVE / WISDOM** — per-org premium (value-based, sold to CEO).
- **PROCURE** — subscription + small transaction fee (aligns price with value/volume).
- **COMPLIANCE** — per-standard module (pay for ISO 27001, add PDPA, add 45001…).
- **DIAGNOSIS / Blueprints** — fixed-fee (consulting + productized IP).

## Go-to-market motion
1. **Prove on BEST** — undeniable reference: 1,356 staff, 256 sites, real fraud/leakage removed.
2. **Sell the fear, deliver the OS** — lead with a Tier-A problem (ghost employees / payroll fraud), land CHECK, expand to the platform.
3. **Channel through consultants** — ISO consultants and accountants already sit in the buying conversation; make them resellers (stream 9).
4. **Vertical-by-vertical** — start with cleaning/FM/security (BEST's world, blueprints ready), then logistics/warehouse/manufacturing.
5. **Land-and-expand** — every product is a door; the OS is the house.

## Unit-economics watch-items
Gross margin is software-grade (high) except Consulting (people-bound). The
strategy intentionally uses low-scalability/high-cash Consulting to fund
high-scalability/recurring SaaS — the classic services-to-product flywheel.
